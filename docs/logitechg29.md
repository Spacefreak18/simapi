# Logitech G29

The Logitech G29 will track rpms with it's leds.

Use this config (or add relevant block to an existing config):


```
configs = ({
    //sim = "default";
    sim = "default";
    car = "default";
    devices = ( {
                    device       = "USB";
                    type         = "Wheel";
                    subtype      = "LogitechG29";}
            );
});

```
